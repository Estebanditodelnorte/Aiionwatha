# Checklist de lancement de site

Verifications essentielles avant la mise en ligne.

- Domaine et SSL : redirections www/non-www, HSTS, certificats valides.
- SEO : balises title/meta, sitemap, robots.txt, schemas essentiels.
- Performance : images compressees, cache CDN, lazy-load medias.
- Accessibilite : labels formulaires, focus visible, contrastes AA.
- Analytics : conversions, exclusions d'IP interne, consentement.
- Sauvegardes : point de restauration et rollback rapide.
