# Budget previsionnel (Excel)

Guide pour le classeur de projections.

Onglets recommandes
- Hypotheses : volumes, prix, saisonnalite, taux de change.
- Ventes : previsions mensuelles par ligne dâ€™affaires.
- Cout des ventes : achats, production, livraisons.
- Charges : salaires, marketing, tech, loyer, frais generaux.
- Tresorerie : encaissements/decaissments, besoins de fonds.
- KPI : marge, CAC, LTV, point mort.

Conseils
- Mettre en forme les cellules editables (bleu) vs calculees (gris).
- Verifier les liaisons avant export PDF.
