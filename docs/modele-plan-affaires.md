# Modele de plan d'affaires (Google Docs)

Structure proposee pour le gabarit Google Docs.

Chapitres
- Resume executif : offre, marche, traction, besoins financiers.
- Marche : segmentation, pain points, analyse concurrence.
- Produit/Service : proposition de valeur, differenciateurs, roadmap.
- Marketing/Ventes : canaux prioritaires, prix, previsions.
- Operations : equipe, processus, partenaires.
- Finances : hypotheses, seuil de rentabilite, besoins en liquidites.

Pieces jointes
- Tableaux financiers relies au budget previsionnel.
- Annexes juridiques (statuts, conventions essentielles).
