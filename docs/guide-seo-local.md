# Guide SEO local (PDF)

Structure recommandée du guide PDF pour optimiser la présence locale.

Sections suggérées
- Audit de base : Google Business Profile, NAP cohérent, catégories.
- Pages locales : balises title/meta, schéma LocalBusiness, avis clients.
- Citations : MRC/annuaires régionaux, niches sectorielles.
- Contenu : FAQ locale, offres géociblées, FAQ schema.
- Suivi : Search Console, Insights GBP, positions locales mobiles.

Livrables
- PDF prêt à télécharger.
- Checklist imprimable en fin de document.
- Liste de 10 liens d’outils gratuits pour la recherche locale.
