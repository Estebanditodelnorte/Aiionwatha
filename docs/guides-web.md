# Guides web

Série de guides pratiques pour lancer et maintenir un site performant et accessible.

Contenu
- Stratégie : objectifs, personas, parcours utilisateur.
- Conception : maquettes responsives, hiérarchie H1–H3, contraste.
- Technique : performance de base, sécurité (HTTPS, headers), backups.
- Gouvernance : rôles, calendrier éditorial, revue trimestrielle.
