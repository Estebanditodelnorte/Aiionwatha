# Resume Knowledge Base

This folder is the internal source of truth for the public Steven Germain Gros-Louis resume pages.

Goals
- Keep a public-safe record of current projects, completed work, skills, credentials, and results.
- Make it easy to update the website resume without rewriting the whole page each time.
- Separate public profile material from working notes and internal strategy docs.

Recommended update workflow
1. Update `master-profile.md` when the positioning, headline, or summary changes.
2. Update `current-projects.md` when a new active mandate starts or priorities shift.
3. Update `completed-projects.md` when a mandate ends or a milestone is shipped.
4. Update `skills.md` when new technical, operational, or advisory capabilities are added.
5. Update `results-and-impact.md` when concrete outcomes, metrics, recognitions, or references are available.

Public page goals
- Show what Steven is working on now.
- Show what he has already delivered.
- Show the skills and operating range behind the work.
- Show concrete results, scale, and context.
