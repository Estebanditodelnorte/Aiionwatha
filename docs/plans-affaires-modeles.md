# Plans d'affaires et modeles

Vue d’ensemble des gabarits financiers et stratégiques disponibles.

Inclus
- Modèle de plan d’affaires (Google Docs).
- Budget prévisionnel (Excel).
- Annexes : analyse de marche, plan de commercialisation, risques.

Utilisation
- Dupliquer les modèles puis adapter sections et hypothèses.
- Garder une page de synthese (one-pager) pour les bailleurs/banques.
- Mettre a jour mensuellement les hypotheses de ventes et de couts.
