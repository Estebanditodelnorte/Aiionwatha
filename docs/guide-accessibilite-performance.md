# Guide accessibilite et performance

Trame pour maintenir un site rapide et inclusif.

Accessibilite
- Navigation clavier, ordre logique des focus.
- Texte : contrastes AA, tailles rem/responsives, pas de texte dans les images.
- Formulaires : labels explicites, messages d’erreur clairs, aria-live.
- Media : sous-titres, transcriptions, description des images.

Performance
- Budget perf : LCP <2.5s, INP <200ms, CLS <0.1.
- Optimisation : minification, preconnect CDN, réduction scripts tiers.
- Images : formats modernes, dimensions explicites, responsive srcset.
- Monitoring : Lighthouse régulier, alertes sur erreurs 4xx/5xx.
