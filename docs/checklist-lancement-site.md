# Checklist de lancement de site

Vérifications clés avant mise en ligne.

- Domaine et SSL : redirections www/non-www, HSTS, certificats valides.
- SEO : balises title/meta, sitemap, robots.txt, schémas essentiels.
- Performance : images compressées, cache CDN, lazyload médias.
- Accessibilité : labels formulaires, focus visible, contrastes AA.
- Analytics : balises de conversion, exclusions d’IP interne, consentement.
- Sauvegardes : point de restauration, rollback rapide.
